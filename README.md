# SecureFleet: Issue and Vulnerability Tracking System for GoCar Ireland

## Project Overview

SecureFleet is an API-based issue and vulnerability tracking system designed for a selected named company, GoCar Ireland.

The system allows IT/security staff to record, view, update, delete, search, filter, sort, and report issues or vulnerabilities affecting company digital assets.

This project uses simulated academic data only and does not represent real GoCar Ireland systems or vulnerabilities.

## Selected Company

GoCar Ireland

## Planned Features

- Create, read, update, and delete company assets
- Create, read, update, and delete issues/vulnerabilities
- Search issues by keyword
- Filter issues by severity and status
- Sort issues by due date
- Validate required fields
- Generate summary reports
- Test API endpoints using Postman
